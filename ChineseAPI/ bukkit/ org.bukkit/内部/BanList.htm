org.bukkit

接口BanList


  • 公共接口BanList
    禁令清单，其中包含一些禁令BanList.Type。

    • 嵌套类摘要

      嵌套类 
      修饰符和类型 接口和说明
      static class  BanList.Type
      表示一个班型，一个BanList可以跟踪。

    • 方法详细信息

      • getBanEntry

        BanEntry  getBanEntry（String  target）
        获取BanEntry由目标。
        参数：
        target - 要搜索的条目参数
        返回：
        相应的条目，如果没有找到则为null

      • addBan

        BanEntry  addBan（String  target
               String  reason，
               Date  maturity，
               String  source）
        添加禁止到此列表。如果存在先前的禁止，则这将更新先前的条目。
        参数：
        target - 禁令的目标
        reason - 禁止的原因，null表示实现默认
        expires - 禁令的到期日期（取消停止），或null表示永远
        source - 禁止来源，null表示实施默认
        返回：
        新创建的禁令的条目或（已更新的）先前禁止的条目

      • getBanEntries

        set < BanEntry > getBanEntries（）
        获取包含每一套BanEntry在列表中。
        返回：
        包含由此列表跟踪的每个条目的不可变集合

      • isBanned

        isBanned（String  target）
        获取如果一个BanEntry目标存在，指示活性禁止状态。
        参数：
        target - 找到的目标
        返回：
        真若BanEntry存在名称，表明积极的禁令状态，否则为false

      • pardon

        void pardon（String  对象）
        从此列表中删除指定的目标，因此指示“未禁止”状态。
        参数：
        target - 要从此列表中删除的目标
