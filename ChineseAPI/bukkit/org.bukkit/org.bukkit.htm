Bukkit的常用类.

  • 接口概要 
    接口 描述
    BanEntry
    在封禁列表的单个条目
    BanList
    一个拥有封禁类型的封禁列表
    BlockChangeDelegate
    处理块的变化。
    Chunk
    代表一个16*256*16的空间
    ChunkSnapshot
    表示一个静态的，线程安全的方块的区块的快照
    Nameable
    代表一个可以被命名的食物
    OfflinePlayer
    代表一个玩家，可以是在线和不在线的
    Server
    执行插件的服务器
    TravelAgent
    TravelAgent用于实体试图使用下界或末地传送门时处理相关的生成和搜索。
    UnsafeValues 已过时
    不受支持，只供内部使用
    World
    代表一个世界，包含了实体，区块，方块
    WorldBorder
    用于处理世界边境的接口
  • 类概要 
    类 描述
    Bukkit
    代表一个 Bukkit 核心, 用来转发 Server 单例的调用.
    Color
    这是一个调色板的容器
    FireworkEffect
    代表一个单独的烟花效果
    FireworkEffect.Builder
    这是一个烟花效果的生成器
    Location
    一个世界的三维位置，包含X，Y，Z，Pitch，Yaw
    Note
    一个单独储存的音符
    Server.Spigot  
    World.Spigot  
    WorldCreator
    代表世界生成器，用于储存世界的配置来创建世界

    枚举 描述
    Achievement
    代表一个成就
    Art
    代表画
    BanList.Type
    封禁列表 的封禁类型
    ChatColor
    代表聊天栏支持的所有颜色
    CoalType
    代表煤的两种类型
    CropState
    一个庄稼的状态
    Difficulty
    代表游戏难度
    DyeColor
    染料和布料支持的所有颜色
    Effect
    药水效果列表
    Effect.Type
    代表药水效果的类型
    EntityEffect
    一个实体能发生的效果列表
    FireworkEffect.Type
    效果的类型或形状
    GameMode
    代表HumanEntity不同种类的游戏模式
    GrassSpecies
    代表不同类型的草丛
    Instrument  
    Material
    所有物品堆材质的枚举
    NetherWartsState  
    Note.Tone
    一个包含音调的枚举
    Particle  
    PortalType
    代表世界中的传送门类型
    Rotation
    代表旋转方向的枚举
    SandstoneType
    代表不同种类的沙石
    SkullType
    代表不同种类的头颅类型
    Sound
    服务器能给玩家发送的声音枚举
    SoundCategory
    声音种类的枚举
    Statistic
    代表被服务器检测的统计数据
    Statistic.Type
    统计数据的类型枚举
    TreeSpecies
    代表不同种类的树木
    TreeType
    树结构的枚举
    Warning.WarningState
    This represents the states that server verbose for warnings may be.
    WeatherType
    各种天气的枚举
    World.Environment
    世界可能加载的环境枚举
    WorldType
    世界类型枚举
